# Vim 使用笔记

这篇文章先记一组自己常用的 Vim 基础配置，重点不是讲原理，而是把最常开、最常改的设置集中放在一起，方便后面直接抄回配置文件。

![img](https://cdn.jsdelivr.net/gh/Amdeo/blogImg/img/vi-vim-cheat-sheet-sch.gif)

## 常用配置

```vim
syntax on
set number
set norelativenumber
set cursorline
set wrap
set showcmd
set wildmenu
set hlsearch
set incsearch
```
